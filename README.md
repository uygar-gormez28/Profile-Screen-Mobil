# Premium Profile Screen (React Native)

Bu proje, React Native kullanılarak geliştirilmiş modern, şık ve premium hissettiren bir Profil Ekranı tasarımıdır.

## Özellikler

- **Gelişmiş Profil Kartı:** Kullanıcı resmi, çevrimiçi durum göstergesi ve profesyonel tipografi ile tasarlanmış profil kartı.
- **Yatay Kaydırılabilir Favoriler:** Kullanıcının favori gönderilerini veya fotoğraflarını listeleyen yatay kaydırılabilir (`ScrollView`) alan.
- **Kişiselleştirilmiş Renkli Butonlar:** Her biri kendi temasına uygun renklere ve arka plan aydınlatmasına sahip, Expo Icons (Ionicons) ile zenginleştirilmiş etkileşimli menü butonları.
- **Dokunma Animasyonları:** `Pressable` bileşeninin yetenekleri kullanılarak, dokunulduğunda hafifçe küçülen ve saydamlaşan dinamik buton efektleri (Scale/Opacity).
- **Esnek Yapı:** Parçalanmış bileşen mimarisi (`ProfileCard`, `ProfileFavorie`, `ProfileButton`) sayesinde kolay okunabilir ve yönetilebilir kod tabanı.

## Kullanılan Teknolojiler

- [React Native](https://reactnative.dev/)
- [Expo Vector Icons](https://docs.expo.dev/guides/icons/) (Ionicons & AntDesign)
- [React Native Size Matters](https://github.com/nirsky/react-native-size-matters) (Farklı ekran boyutlarında uyumlu tasarım için)

## Kurulum ve Çalıştırma

Projeyi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. **Bağımlılıkları Yükleyin:**

   ```bash
   npm install
   ```

2. **Uygulamayı Başlatın:**

   ```bash
   npx expo start
   ```

3. Çıkan ekranda **a** tuşuna basarak Android emülatöründe, **i** tuşuna basarak iOS simülatöründe veya telefonunuzdan Expo Go uygulaması ile QR kodu okutarak test edebilirsiniz.
